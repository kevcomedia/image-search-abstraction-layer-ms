Image Search Abstraction Layer

Image search

/search?q=searchTerm[&offset=number]

Examples

/search?q=tuna sandwich

/search?q=funny tuna?offset=2

Output

An array of ten results with the following details:

url
A URL that points to the image.
alt
The title of the page the image appears in.
context
A URL that points to the page that contains the image.

Recent searches

/recent

Output

An array of the last ten searches and their timestamps.

query
The search term.
timestamp
The date and time the search was made.
